Software Projects

NCAA Player Search (SQL & Java):

Using a database of thorough NCAA statistics, our team built a GUI application which had features such as a player search, calculated the statistical rival of a team, and found victory chains. One of my major contributions was a homefield advantage calculation algorithm, which evaluated a team’s relative performance at home and overall which objectively determined if a team has a homefield advantage.

Nine To Five Job Search Site (HTML, CSS, Javascript):

Compiles data from job boards, glassdoor, and LinkedIn to provide a more catered search for internships, specifically within software engineering. Additionally, this site sources a list of intern program cancellations due to COVID. I built most of the UI/UX of the site, while also building some of the API requests.

Question Ball (HTML, CSS, Javascript):

Simple question ball site used by college orientation leaders to break the ice with freshmen during new online programs. Generates a random question from a user defined array.
